# 🏆 Gemini 3 Hackathon - Winning Submission Checklist

Even if the video is "average," you can win on **Utility**, **Creativity**, and **Technical Execution**. Use this guide to polish your Devpost submission.

## 1. 🖼️ The "Hero" Cover Image (Crucial)
*Judges click on projects with good thumbnails.*
*   **Do not** use a generic logo.
*   **Do**: Take a high-res screenshot of the **"Decision Triangle"** or the **"Thinking Panel"** in action.
*   **Overlay Text**: Add big, bold text: *"AI-Powered Energy Auditing"*.

## 2. 📝 The "Story" (Copy-Paste from SUBMISSION_MEMO.md)
We have updated your `SUBMISSION_MEMO.md` to map directly to Devpost fields:

*   **Inspiration**: Use the "5 cars" analogy. It hits hard emotionally.
*   **What it does**: Focus on the *Agentic* workflow (See -> Search -> Execute -> Optimize).
*   **How we built it**: Highlight the "Hybrid Grounding" (AST + LLM). This shows technical depth.

## 3. 📸 Image Gallery (Compensate for Video)
Since the video is simple, overload the image gallery with annotated screenshots:
1.  **Multimodal Input**: Show the "Hand-drawn sketch" side-by-side with the "detected bottleneck".
2.  **Agent Reasoning**: Screenshot the "Thinking Panel" showing the Google Search query for "NVIDIA B200 TDP". *This proves it's real.*
3.  **Code Exec**: Screenshot the "Sandbox" output verifying the math.
4.  **Before/After**: A split comparison of the "Original Code" vs "Green Code".

## 4. 🧪 "Try It Out" Instructions
*Make it impossible to fail.*
*   If you have a live link (Google AI Studio or Vercel), put it at the **very top**.
*   Provide a "Test Prompt" so judges don't have to think.
    *   *Example*: "Paste the sample ResNet-50 code from the dropdown and click Analyze."

## 5. 💻 The "Magic" Code Snippet
Devpost often asks for a code snippet. Use the configuration block I added to the `README.md` (under "Gemini 3 Agentic Features"). It shows you are using the *newest* features:
*   `googleSearch: {}`
*   `codeExecution: {}`
*   `thinkingBudget: 1024`

## 6. 🚀 Final Polish
*   **Tagging**: Ensure you used tags like `Gemini 3`, `Sustainability`, `Agent`, `React`.
*   **Team**: If you have teammates, make sure they are all linked.

---
**Strategy Summary**:
Move the focus from "Production Value" (Video) to **"Engineering Value"** (Docs & Features). Judges love a tool that feels like a real, useful utility rather than just a flashy demo.
